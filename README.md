# 📝 InkFlow

A full-stack real-time collaborative document editor. InkFlow allows multiple users to create, edit, and share rich-text documents with role-based access control and live collaboration.

## Features

- 🔐 JWT Authentication
- 📄 Create, edit, and delete documents
- ✍️ Rich text editing using TipTap
- ⚡ Real-time collaboration with Socket.IO
- 👥 Online collaborator presence
- 🤝 Share documents via email (to registered users)
- 🔒 Role-based permissions (Owner / Editor / Viewer)
- 💾 Debounced auto-save
- 🌐 Fully deployed on Vercel + Render
